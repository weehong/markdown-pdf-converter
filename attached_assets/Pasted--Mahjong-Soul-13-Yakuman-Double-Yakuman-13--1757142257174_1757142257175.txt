在《雀魂》（Mahjong Soul）中，達到 13番以上 的牌型，通常是**役滿（Yakuman 或 兩倍役滿（Double Yakuman）**。以下是各種可以達成 **13番或以上** 的牌型與其組合情況：

---

## 🀄️ 一、單一役滿牌型（13番）

以下每個役滿都單獨算13番：

| 牌型名稱         | 日文              | 條件說明                         |
| ------------ | --------------- | ---------------------------- |
| 天和           | 天和（てんほう）        | 莊家第一巡自摸和牌（無鳴牌）               |
| 地和           | 地和（ちーほう）        | 門前自摸第一巡（副露者除外）               |
| 四暗刻          | 四暗刻（すーあんこう）     | 四組暗刻＋一對（門前限定）                |
| 四暗刻單騎（兩倍役滿）  | 四暗刻単騎           | 四暗刻＋單騎等待（門前）→計為**兩倍役滿**      |
| 大三元          | 大三元（だいさんげん）     | 三元牌（白發中）三刻                   |
| 字一色          | 字一色（ついいーそー）     | 全部由字牌構成                      |
| 小四喜          | 小四喜（しょうすーしー）    | 三風刻＋一風對                      |
| 大四喜（兩倍役滿）    | 大四喜（だいすーしー）     | 四風刻（東南西北各一刻）→**兩倍役滿**        |
| 清老頭          | 清老頭（ちんろーとう）     | 所有牌為1或9（萬索筒）                 |
| 九蓮寶燈         | 九蓮宝燈（ちゅうれんぽうとう） | 同一花色：1112345678999 + 任一張（門前） |
| 純正九蓮寶燈（兩倍役滿） | 純正九蓮宝燈          | 九蓮單騎胡 → 兩倍役滿             |
| 綠一色          | 緑一色（りゅーいーそー）    | 全為綠色牌（發 + 23468索）            |
| 清一色國士無雙（十三么） | 国士無双（こくしむそう）    | 13種么九牌＋任一對（門前）               |
| 十三么單騎（兩倍役滿）  | 国士無双十三面待ち       | 十三么單騎等待 → 兩倍役滿           |
| 四槓子          | 四槓子（すーかんつ）      | 四個槓（門前可、副露可）                 |

---

## 🀄️ 二、複數役滿組合（同時達成多個役滿）

有些牌型可 **組合多個役滿**，達成 **26番以上**，例如：

| 組合說明                       | 番數                |
| -------------------------- | ----------------- |
| 大四喜 + 字一色                  | 26番               |
| 四暗刻單騎 + 純正九蓮               | 26番               |
| 四槓子 + 字一色                  | 26番               |
| 國士無雙十三面待ち + 字一色（有爭議但部分規則可） | 26番               |
| 大三元 + 字一色                  | 26番               |
| 綠一色 + 發碰 + 混一色             | **僅計為綠一色役滿**，不能疊加 |

---

## 🀄️ 三、其他達到13番以上的牌型組合（非役滿）

雖然比較少見，但**多種高番役組合**也可超過13番（**非役滿情況**），例如：

| 組合                                      | 番數           |
| --------------------------------------- | ------------ |
| 四暗刻（13番）+ 赤5（赤ドラ）                       | 14番          |
| 三倍滿組合（例如：清一色 + 一氣通貫 + 平和 + 立直 + 自摸 + 赤） | 可超過13番，但不算役滿 |

但在雀魂中，一旦牌型達到**役滿**，就不再用一般番數計算，而是**直接算役滿倍數**。

---

## 🀄️ 四、兩倍役滿（雀魂支援）

雀魂目前支援以下 **兩倍役滿**：

| 兩倍役滿名稱 | 說明        |
| ------ | --------- |
| 四暗刻單騎  | 單騎胡四暗刻    |
| 大四喜    | 四風刻齊全     |
| 純正九蓮寶燈 | 九蓮單騎胡     |
| 十三么單騎  | 國士無雙十三面待ち |

---

## 🧮 備註

* 雀魂使用日本麻將規則（含赤牌、不使用槍槓四槓散了等），某些情況依房間設定略有不同。
* 雀魂目前**不支援三倍役滿或四倍役滿**，最高為兩倍役滿（26番）。
* 實戰中能胡到役滿已非常罕見，兩倍役滿更是極稀有。
